Política de Privacidade

Coleta de Dados

Os dados coletados dos usuários incluem informações como nome, email, dados de contato, estado, cidade e cidade

Utilização de Dados

Os dados dos usuários são utilizados para criar e gerenciar contas de usuário, processar transações, personalizar a experiência do usuário na plataforma, fornecer suporte ao cliente, análise de dados para melhorar os serviços oferecidos e oferecer sugestões personalizadas com base no tipo e foco do negócio do usuário.

Armazenamento de Dados

Os dados dos usuários são armazenados em um banco de dados na nuvem, com medidas de segurança adequadas para proteger contra acesso não autorizado ou uso indevido. As informações são armazenadas de forma criptografada e são acessíveis apenas por pessoal autorizado.

Compartilhamento de Dados

Atualmente, não há compartilhamento de dados com terceiros. No entanto, no futuro, caso haja necessidade de compartilhamento de dados para prestação de serviços adicionais, os usuários serão notificados e terão a opção de consentir antes que seus dados sejam compartilhados. Medidas serão tomadas para garantir a proteção dos dados compartilhados.

Cookies e Tecnologias de Rastreamento

A plataforma pode utilizar cookies ou outras tecnologias de rastreamento indiretamente, seja por meio de serviços de API de terceiros ou funcionalidades das próprias tecnologias utilizadas. Essas tecnologias são utilizadas para melhorar a experiência do usuário e coletar informações para análise, mas os usuários têm a opção de gerenciar suas preferências de cookies.

Direitos dos Usuários

Os usuários têm o direito de acessar, corrigir, excluir ou limitar o uso de seus dados pessoais a qualquer momento, conforme previsto pela legislação de proteção de dados aplicável, como a LGPD. Esses direitos podem ser exercidos por meio da área administrativa do usuário na plataforma.

Alterações na Política de Privacidade

A política de privacidade pode ser atualizada periodicamente para refletir mudanças nas práticas de privacidade da plataforma ou em conformidade com as leis aplicáveis. Os usuários serão notificados sobre quaisquer alterações na política e serão encorajados a revisar as atualizações. O uso contínuo da plataforma após as alterações na política será considerado como consentimento às novas práticas de privacidade.

Contato

Para dúvidas ou preocupações sobre a Política de Privacidade ou o tratamento de dados, os usuários podem entrar em contato com a plataforma enviando um e-mail para: afriendforlifepet@gmail.com
